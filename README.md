






# Bradesco Course
   I developed this project on the Bradesco course. The purpose of it is to learn how to create a simple website using HTML,
CSS and JavaScript that switches the website theme with a button, depending on your current.

### Objetives:

* 1st: Learn HTML structure syntax and how to link css and JavaScript file.
* 2st: Learn how to modify the body with CSS tools.
* 3st: Learn how to manipulate HTML elements to switch the theme using:
  
  * ***Docment Object Model***,
  * ***Callback functions***,
  * ***If-Statement***.
