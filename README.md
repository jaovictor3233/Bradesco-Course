

# Bradesco Course : Theme Switcher 

   I developed this project on the Bradesco course. The purpose of it is to learn how to create a simple website using HTML,
CSS and JavaScript that switches the theme with a button, depending on your current section.

### Objetives : 

* 1st: Learn HTML structure syntax and how to link css and JavaScript file.
* 2nd: Learn how to modify the `body` with CSS tools.
* 3rd: Learn how to manipulate HTML elements to switch the theme using:

  * ***Document Object Model***,
  * ***Callback functions***,
  * ***If-Statement***.
